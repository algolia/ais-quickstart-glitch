Algolia Quickstart

Algolia is a search API that helps you add a search bar to websites and applications. In this tutorial you will build an Algolia search step-by-step starting with indexing and ending with search. In just 5 minutes, you'll have learned how search works and have a place to search your favorite actors.

Remix Instructions

😎 Welcome! We are so happy you are here! Try Algolia in 6 simple steps 👯‍

  • 🎏 Step 1: Remix this app

    Click this button, which creates your own live copy of this app that you modify and extend right in your browser.

  • 🔑 Step 2: Get your Algolia API key

    Head to Algolia for API keys. You'll want to copy your admin key, app id and search only key.
    Once this is complete, add them to your .env file so your application can run.

  • 🗃 Step 3: Prepare JSON data for Algolia

    We need to create an index with Algolia with some data we want to search. Head to your server.js file at the root of your project to set your data to index. Start by commenting out some code and follow along with instructions.

    Check data structure
  • ☁️ Step 4: Upload data to Algolia index

    Go to server/helpers/algolia.js and comment in the function calling our data.

    Upload data
  • 🛠 Step 5: Configure index to be searched

    Go to server/helpers/algolia.js and comment in the setSettings function.

    Configure Index
  • ✨ Step 6: Add search widgets to UI

    Go to public/search.js and comment in the widgets.searchBox and widgets.hits

    View your data

Success!

  • 🖥 You've completed the tutorial, you've learned how to: 🖥
  • 💫 Index data
  • 💪 Configure an index
  • 🔎 Add widgets to your UI
