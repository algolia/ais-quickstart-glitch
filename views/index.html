Algolia Quickstart

Create a fully functioning instant search for your website by going through the steps below with Algolia. We will waklk you through getting your api keys, sending data to an Algolia index, creating your search settings and ending with instant search ready to go!

Remix Instructions

😎 Welcome! We are so happy you are here! Try Algolia in 4 simple steps 👯‍

  • 🎏 Step 1: Remixing

    Remix this app

  • 🔑 Step 2: Pre-requisites

    Head to Algolia for API keys, this should take you about 3 minutes to set up. You'll want to copy your admin key, app id and search only key.
    Once this is complete, add them to your .env file so your application can run.
    We need to create an index with Algolia with some data we want to search. Head to your server.js file at the root of your project to set your data to index. Start by commenting out some code and follow along with instructions.

  • 🗃 Step 3: Indexing and Settings

    We need to tell Algolia how we want our data to be indexed and searched. Head to your server/helpers/algolia.js file and follow along with instructions.

  • ✨ Step 4: Instant Search

    Make your /search your homepage and remove this index page!
